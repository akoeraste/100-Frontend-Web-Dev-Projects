# 023 - Image Slider / Carousel

Cycle through images automatically or manually with navigation arrows, dots, auto-play toggle, and keyboard support.

## Preview

![Image Slider](../images/023.png)

## Features

- **5 landscape images** with captions
- **Auto-play** with pause/play toggle
- **Prev/Next arrows** and **dot indicators**
- **Keyboard navigation** — Left/Right arrows
- **Slide counter** (1/5, 2/5, etc.)
- **Smooth CSS transitions**
