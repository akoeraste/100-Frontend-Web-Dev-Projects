About This Demo

This project demonstrates the sticky header pattern used in modern websites. As you scroll, the header transitions smoothly — shrinking in height, gaining a frosted glass background, and adding a subtle box-shadow. The effect is achieved with pure JavaScript scroll detection and CSS transitions.

The header uses position: fixed and JavaScript adds a .scrolled class when the user scrolls past a threshold. All visual changes are handled by CSS transitions for smooth 60fps animations.
