# 044 - Drawing App

Freeform drawing canvas with pen and eraser tools, color picker, brush size control, undo, and image download.

## Preview

![Drawing App](../images/044.png)

## Features

- **Freeform drawing** on an HTML5 canvas
- **Pen and eraser** tool toggle
- **Color picker** for any color
- **Brush size slider** with live label
- **Undo** up to 30 steps
- **Clear canvas** in one click
- **Download** your drawing as a PNG
- **Touch support** for mobile/tablet drawing
- **Responsive** canvas that resizes with the window
