# 015 - Image Gallery

A filterable image gallery with hover overlays, click-to-open lightbox, and keyboard navigation.

## Preview

![Image Gallery](../images/015.png)

## Features

- **Category filters** — All, Nature, City, Abstract with smooth show/hide
- **12 curated images** from Unsplash
- **Hover overlay** with image title and zoom effect
- **Full lightbox** — click any image to open a high-res view
- **Lightbox navigation** — prev/next arrows + keyboard (Left, Right, Escape)
- **Click backdrop to close**
- **Responsive grid** that adapts from 3 columns to 1
