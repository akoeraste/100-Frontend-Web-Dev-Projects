# 048 - Emoji Rating

Rate items with expressive emoji feedback on a 1–5 scale. See an average score and summary of all ratings.

## Preview

![Emoji Rating](../images/048.png)

## Features

- **5-point emoji scale** — Terrible, Bad, Okay, Good, Amazing
- **5 ratable items** — Customer Service, Product Quality, Delivery Speed, Value for Money, Overall Experience
- **Visual feedback** — selected emoji scales up, others dim
- **Live summary** with average score and corresponding emoji
- **Rating breakdown** listing each item's emoji
- **Reset all** button to clear ratings
- **Responsive** layout
