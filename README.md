# 100 Projects of Web Development Challenge

> Build one real-world project every day for 100 days using HTML, CSS, JavaScript, and React.

---

## About

A personal challenge to sharpen front-end skills by building 100 mini-projects from scratch. Each day tackles a unique concept — from layout fundamentals to API integrations, async patterns, and full React apps.

## Structure

| Phase | Days | Focus | Key Skills |
|-------|------|-------|------------|
| **Level 1** | 1 -- 25 | HTML, CSS & Basic JS | Layout, forms, DOM basics, simple logic |
| **Level 2** | 26 -- 50 | Interactive JavaScript | DOM manipulation, browser APIs, localStorage |
| **Level 3** | 51 -- 75 | Advanced JS & APIs | Async/await, REST APIs, charts, ML in the browser |
| **Level 4** | 76 -- 100 | React Projects | Components, hooks, context, Firebase, full apps |

---

## Tech Stack

- **HTML5 & CSS3** — Flexbox, Grid, animations, responsive design
- **JavaScript (ES6+)** — DOM, async, fetch, Web APIs
- **React** — Hooks, Context API, component architecture
- **Libraries** — Chart.js, TensorFlow.js, Firebase (in later projects)

## How to Use

```bash
git clone https://github.com/akoeraste/100dayswebchallenge.git
cd 100dayswebchallenge
```

- **HTML/CSS/JS projects** — open `index.html` in a browser
- **React projects** — `npm install && npm start`

Each `Day-XX/` folder is a standalone project with its own code and assets.
