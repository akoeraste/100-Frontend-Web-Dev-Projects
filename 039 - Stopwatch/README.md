# 039 - Stopwatch

Precise stopwatch with start, pause, reset, and lap tracking. Best and worst laps are highlighted.

## Preview

![Stopwatch](../images/039.png)

## Features

- **Centisecond precision** display (HH:MM:SS.ms) using requestAnimationFrame
- **Start / Pause** toggle with color-coded button (green = start, amber = pause)
- **Lap tracking** with split times and cumulative totals
- **Best lap** highlighted in green, **worst** in red
- **Reset** clears time and all laps
- **Clear laps** keeps the timer running but clears the list
- **Monospace font** for consistent digit width
- **Responsive** layout
