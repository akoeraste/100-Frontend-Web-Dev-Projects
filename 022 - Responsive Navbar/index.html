Responsive Navbar

Resize the window or click the hamburger on mobile to see the navigation adapt. Features animated toggle, glassmorphism, scroll effect, and active link highlighting.

About

This navbar features a glassmorphism background on scroll, smooth mobile slide-in menu, animated hamburger toggle, and active section tracking as you scroll through the page.

Services

The responsive design breaks from a horizontal layout to a full-screen mobile overlay at 768px. Each link has an icon and hover animation. The CTA button sits in the nav for quick access.

Portfolio

Built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no libraries beyond Font Awesome for icons and Google Fonts for typography. Clean and lightweight.

Contact

The active nav link updates automatically as you scroll between sections using Intersection Observer. Try scrolling through the page and watch the nav links highlight.
