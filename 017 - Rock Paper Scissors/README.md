# 017 - Rock Paper Scissors

Play against the computer with emoji picks, win/lose/draw tracking, and color-coded results.

## Preview

![Rock Paper Scissors](../images/017.png)

## Features

- **3 choice buttons** with emoji and hover animations
- **Scoreboard** — You vs CPU with color-coded scores
- **Arena** — shows both picks with glowing borders (green=win, red=lose, amber=draw)
- **Result badge** with color-coded feedback
- **Reset** to clear scores
