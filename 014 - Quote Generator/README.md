# 014 - Quote Generator

Click for a fresh motivational quote with smooth fade transitions. Copy to clipboard or share directly on X/Twitter.

## Preview

![Quote Generator](../images/014.png)

## Features

- **30 curated quotes** from famous thinkers and leaders
- **Fade transition** between quotes
- **No repeat** — never shows the same quote twice in a row
- **Copy to clipboard** with toast notification
- **Share on X/Twitter** — pre-formatted tweet link
- **Keyboard support** — press Space for a new quote
- **Elegant serif typography** for the quote text
